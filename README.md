
# Next.js + Builder.io + IA (Professor Lucas)

## Como usar:
1. Suba este projeto no GitHub.
2. Vá na Vercel → Import Project.
3. Configure as variáveis:
   - NEXT_PUBLIC_BUILDER_API_KEY = sua public key do Builder
   - OPENAI_API_KEY = sua chave secreta da OpenAI
4. Deploy.

Todas as páginas criadas no Builder (modelo "page") vão funcionar automaticamente.
